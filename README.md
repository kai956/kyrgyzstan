# Kyrgyzstan Presentation Website

A "wow" presentation website for Kyrgyzstan, featuring parallax scrolling, interactive timelines, and immersive animations.

## Tech Stack
- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Fonts**: Instrument Serif & Instrument Sans

## Getting Started

### Prerequisites
- Node.js >= 20.9.0

### Installation
1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser.

## Structure
- `components/`: Reusable UI components for each slide.
- `app/`: Next.js app router pages and layouts.
- `lib/`: Utility functions.

## Features
- **Home**: Parallax hero section.
- **History**: Horizontal scrolling timeline.
- **Politics**: Vertical expandable timeline.
- **Mythology**: Interactive storybook cards.
- **Regions**: Interactive grid.
- **Culture**: Split-screen comparisons.
- **Sports**: Dynamic grid.
- **More**: Tabbed sections for Cuisine, Music, Crafts.
